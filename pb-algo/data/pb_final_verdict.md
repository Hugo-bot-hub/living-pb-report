### 권장 기준: **재구매율 (Rebuy Rate)**

**정의**: 90일 윈도우 내 동일 상품을 2회 이상 구매한 유저 수 / 총 구매자 수

**왜 이 변수인가**:
1. **AUC 0.756** — 12개 변수 중 가장 높은 변별력 (PB vs 3P 식별력)

3. **p-value 2e-13** — 통계적 매우 유의
4. **법적 안전**: 모든 셀러에 동일 측정, 자체 브랜드 식별 무관, 후생 의미 명확 (사용 후 만족 → 재구매 → 품질 신호)

**보완 변수**: 배송정시율 (PB 99.49% vs 3P 95.15%) — 두 번째로 PB 우수. 단, 같이 쓰면 1위 비율이 오히려 약간 감소 (재구매 weight 희석)

## 4. 왜 PB 전체가 1위에 못 오르는가

### 진단

1. **가구 카테 (PB 강점)**: 재구매율 PB 8.5% vs 3P 3.3% (2.6x 압도적 차이)
   - 모듈러 시스템 가구 → 색상/사이즈 추가 구매 + 시스템 확장 구매
   - studio/refine/basic 라인업으로 일관된 라인 구매 유도

2. **패브릭 카테 (PB 약점)**: 재구매율 PB 3.2% vs 3P 3.0% (차이 미미, p=0.23)
   - 침구/이불은 본질적 재구매 주기 1년+
   - 90일 윈도우 짧음 → 패브릭은 의미 있는 차이 미검출

3. **생활용품 카테 (PB 약점)**: PB 1개씩 (수건/욕실잡화/실내화)
   - 신상 출시 → 누적 리뷰/노출 부족
   - 카테 TOP25 3P (메가히트 상품)와 비교 → 진입 장벽 큼

### 카테별 사유
| 카테 | PB 랭킹 | 1위 못 하는 사유 |
|---|---|---|
| 이불/이불커버 (6위) | 재구매 주기 길어 90일 윈도우에서 신호 약함 |
| 패드/스프레드 (10위) | 패브릭 신상 + 시즌제 |
| 전신거울 (11위) | PB 1개, 재구매율 자체 매우 낮음 |
| 커튼 (17위) | 한 번 사면 거의 안 바꿈 |
| 드레스룸 (18위) | 마찬가지로 평생재 |
| 욕실잡화 (21위) | 욕실화 단일 PB, 작은 카테 |
| 실내화/거실화 (24위) | 신상 + 카테 인지도 부족 |

## 5. 권고 사항

### 5-1. 단기 (현재 데이터로 즉시 가능)
- **알고리즘 변수**: 재구매율 단독 또는 재구매율 0.7 + 배송정시율 0.3
- **결과 보장**: 가구 11개 카테에서 PB 자동 1위 진입
- **법적 근거**: AUC 0.756, p=2e-13, 모든 셀러 동일 측정

### 5-2. 중기 (3~6개월 후 재시도)
- **윈도우 180일+ 확장**: 패브릭 재구매 주기 반영 → 이불/패드 1위 진입 가능성
- **신상 가중치 보정**: 출시 30일 미만 상품은 별도 normalize → 신상 PB 보호

### 5-3. 장기 (현재 데이터 한계)
- **커튼/드레스룸/생활용품 등 single-purchase 카테**: 재구매율 변수로는 영구 1위 불가
- 대안: 별도 인기도 보정 / 신상 부스트 (단, 법적 정당성 별도 검토)

## 6. 최종 답변 (User 질문에 1:1 대응)

### Q1. 어떤 변수를 기준으로 삼아야 하는가?
**A: 재구매율 (rebuy_rate, 90일 윈도우)**
- 12개 변수 중 유일하게 통계적 유의(p=2e-13) + 강한 효과(d=1.29) + PB 우수
- 측정 정의서, SQL, 모든 셀러 동일 적용 검증 완료

### Q2. PB 전체가 1위에 올 수 있는가?
**A: No.** 재구매율 단독 적용 시 PB 76개 중 **12개(15.8%)만 카테 1위**.
- **가능**: 11개 가구 카테 PB 100% 1위 진입 (서랍/수납침대/식탁/옷장/책상/책장/소파/의자/침대/화장대/매트리스)
- **불가**: 패브릭(이불/패드/베개), 생활용품(수건/욕실/실내화), 일부 가구(전신거울/사이드테이블/드레스룸) — 18개 카테에서 1위 진입 불가

### Q3. 1위가 안 되는 카테는?
**A:** 패브릭 8개 카테 + 생활용품 3개 + 가구 일부 (전신거울/사이드/드레스룸/책상서랍장/매트/본넬매트리스) = **18개 카테**
- 사유: 재구매 주기 1년+ (패브릭) / 평생재 (커튼·드레스룸) / 신상 + 작은 카테 (생활용품)
